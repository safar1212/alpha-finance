```
  Ruby
```

### Setup
Clone the repository and change the directory to portfolio

``` 
  `$ git clone 'https://github.com/safar1212/alpha-finance'`

  `$ cd alpha-finance`

  `$ bundle install`

  `$ Rails.create()`

  `$ Ruby bin/rails db:migrate`

  `$ http://localhost/3000`

```
